# Café Expresso

## Descrição

O Café Expresso é um sistema de autoatendimento para cafeterias de pequeno porte.
O objetivo é permitir que clientes realizem seus pedidos de forma digital, reduzindo filas e agilizando o atendimento.

---

## Objetivo

* Permitir a criação de pedidos
* Calcular automaticamente o valor total
* Controlar o status dos pedidos

---

## Funcionalidades

* Cadastro de produtos (nome e preço)
* Adição de itens ao pedido
* Cálculo do valor total
* Controle de status:

  * PENDENTE
  * PAGO
  * EM_PREPARO
  * FINALIZADO

---

## Estrutura Inicial

O sistema será composto por:

* Produto
* ItemPedido
* Pedido
* StatusPedido (enum)

---
